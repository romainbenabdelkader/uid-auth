# Contributing

UID_AUTH is an open standard for a sovereign, verifiable identifier for
creative works (origin, integrity and AI/TDM rights signals).

Contributions are welcome via GitHub Issues and Pull Requests.

## Scope

In scope:
- the UID_AUTH identifier format and JSON / JSON-LD schema
- origin, integrity and AI/TDM rights signals
- examples and validation

Out of scope:
- watermarking, fingerprinting, content detection
- DRM or rights enforcement

## How to contribute

- Open an Issue to discuss ideas or report problems
- Submit a Pull Request for concrete proposals
- Keep examples consistent with the schema:
  `python3 scripts/validate_examples.py`
